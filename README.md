# QRify Chrome Extension

## Overview

QRify is a powerful Chrome extension designed to simplify the process of generating QR codes directly from your browser. With just one click, QRify allows you to create QR codes for the current tab or convert any specified URL or text into a shareable QR code effortlessly. Whether you want to quickly share a website or save information for later, QRify has got you covered.

## Features

### 1. One-Click QR Code Generation

Generate a QR code for the currently open website with just one click. This feature streamlines the process, making it convenient to share links or store information.

### 2. URL and Text Conversion

Easily transform any specified URL or text into a QR code. Simply input the desired URL or text, and QRify will generate a QR code that can be shared, printed, or saved as an image.

### 3. Quick Accessibility

The extension is accessible directly from the Chrome toolbar, ensuring that QR code generation is always just a click away. This seamless integration enhances the user experience and makes QRify a convenient tool for everyday use.

## Installation

To install QRify, follow these steps:

1. **Enable Developer Mode:**
   - Open the Chrome browser.
   - Go to `chrome://extensions/`.
   - Enable "Developer mode" in the top right corner.

2. **Load the Extension:**
   - Download the QRify extension files from the [GitHub repository](https://github.com/sudipmahato432/QR-Code-generator-Chrome-extention.git).
   - Extract the downloaded ZIP file to a location on your computer.
   - In `chrome://extensions/`, click on "Load unpacked" and select the extracted folder.

Once loaded, the QRify icon will appear in the Chrome toolbar, ready for use.

## How to Use

1. **One-Click QR Code for the Current Tab:**
   - Click on the QRify icon in the Chrome toolbar.
   - The extension will generate a QR code for the currently open website.

2. **Convert URL or Text to QR Code:**
   - Click on the QRify icon in the Chrome toolbar.
   - Enter the desired URL or text in the provided input field.
   - Click the "Generate" button to create the QR code.

3. **Save or Share the QR Code:**
   - Right-click on the generated QR code.
   - Choose the appropriate option to save the image or copy the QR code for sharing.
